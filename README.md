# Smartphone Addiction Prediction Dashboard

A modern, professional web application for predicting and analyzing smartphone addiction levels using machine learning.

## 🎨 Design Features

- **Glassmorphism UI** with soft shadows and blur effects
- **Professional gradient color scheme** (Indigo/Purple/Cyan)
- **Inter font family** for clean, modern typography
- **Responsive layout** that works on all devices
- **Smooth animations** and hover effects
- **Healthcare-focused aesthetic** - supportive, not judgmental

## 📁 Project Structure

```
smartphone-addiction-dashboard/
├── index.html              # Login page
├── dashboard.html          # Main dashboard with analytics
├── prediction.html         # Prediction form
├── result.html            # Results display
├── src/
│   ├── css/
│   │   └── styles.css     # All styling
│   ├── js/
│   │   ├── dashboard.js   # Chart configurations
│   │   └── prediction.js  # Prediction logic
│   └── assets/            # Images and icons
└── README.md
```

## 🚀 Features

### Login Page
- Clean, centered layout
- Digital wellbeing illustration
- Email/password authentication

### Dashboard
- **Risk Score Card** - Large visual display of addiction level
- **Usage Statistics** - Screen time, unlocks, night usage
- **Weekly Trend Chart** - Line graph showing usage patterns
- **Category Breakdown** - Doughnut chart for app categories
- **Risk Warnings** - Alert panel with actionable insights
- **Personalized Tips** - Recommendations for improvement

### Prediction Form
- Input fields for usage metrics
- Real-time validation
- Loading states
- Professional form design

### Results Page
- Addiction level classification (Low/Medium/High)
- Confidence score display
- Detailed factor analysis
- Improvement suggestions with priority levels
- Progress gauge visualization

## 🎯 Usage

1. Open `index.html` in a web browser
2. Click "Sign In" to access the dashboard
3. Navigate to "Prediction" to enter usage data
4. View detailed results and recommendations
